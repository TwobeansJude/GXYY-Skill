# GXYY 梗创作工坊

一个基于「GXYY」（恭喜娅娅）网络迷因的 **Claude Code Skill**，将拼音缩写文字游戏系统化为可复用的 AI 创意工具。

## 这是什么

GXYY 是中文互联网上广为人知的拼音文字游戏——用首字母为 **G、X、Y、Y** 的四个汉字，去翻译、总结、解构任何内容。本 Skill 将这个玩法变成 Claude Code 的一项技能，支持三种创作模式：

| 模式 | 输入 | 输出 | 示例 |
|------|------|------|------|
| 🎵 **翻译模式** | 英文单词/歌名 | G-X-Y-Y 四字翻译 + 创作思路 | `the cure` → **高新医院** |
| 📜 **总结模式** | 中文散文/场景描写 | 一组 G-X-Y-Y 短语概括意境 | 写景散文 → **观溪遥雁、孤霄悠扬等**… |
| 🎭 **社交玩梗** | 人名/事件关键词 | G-X-Y-Y 圈内黑话 + 背景弹幕 | 宫脇咲良+张元英 → **宫脇元英** |

另有 **自由发挥模式**——直接随机生成好笑的 GXYY 变体。

## 快速开始

### 安装

将 `gxyy/` 文件夹复制到你的 Claude Code skills 目录：

**用户级安装**（所有项目可用）：
```bash
cp -r gxyy/ ~/.claude/skills/gxyy/
```

**项目级安装**（仅当前项目）：
```bash
cp -r gxyy/ your-project/.claude/skills/gxyy/
```

### 使用

在 Claude Code 中输入以下任意触发词即可：

```
GXYY
gxyy
恭喜娅娅
帮我GXYY翻译 the cure
用GXYY总结这段文字：（粘贴你的文字）
玩个GXYY梗
/翻译 expectations
/搞笑
/再来
```

### 指令参考

| 指令 | 功能 |
|------|------|
| `/翻译 <英文>` | 强制使用翻译模式 |
| `/总结 <文段>` | 强制使用总结模式 |
| `/玩梗 <人名/事件>` | 强制使用社交玩梗模式 |
| `/文艺` | 输出风格偏向古风雅致 |
| `/搞笑` | 输出风格偏向荒诞幽默 |
| `/反转` | 输出风格偏向意外反差 |
| `/再来` | 换一个角度重新生成 |
| `/串联` | 将多个翻译串成顺口溜 |

## 示例

### 翻译模式
```
> 帮我GXYY翻译 expectations

🎵 英文原词：expectations
📝 GXYY翻译：高悬娅愿
💡 创作思路：「期望」→「悬着的愿望」+ 嵌入「娅」字，
   双关「悬而未决」，三层含义一次到位。
```

### 总结模式
```
> 用GXYY总结：深夜的城市，霓虹灯在雨水中拉出长长的倒影……

🔍 原文核心意象：霓虹(光)、深夜(夜)、雨水(雨)、倒影(影)、天桥、烟、记忆、风衣
📜 GXYY总结：孤心夜影、光线烟雨、高悬夜影
💡 总结思路：提取「孤/光—心/线—夜/烟—影/雨」四组对照，
   全部严格遵循 G-X-Y-Y 首字母，以孤寂与光影的对照呼应原文的城市疏离感。
   孤心夜影(G-X-Y-Y) = 孤独的内心 + 夜晚的倒影
   光线烟雨(G-X-Y-Y) = 霓虹光线 + 雨中的烟雾
   高悬夜影(G-X-Y-Y) = 天桥高悬 + 夜色剪影
```
